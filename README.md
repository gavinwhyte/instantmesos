### Instant mesos like instant noodles.


# Please ensure you have boot2docker enstalled and started.


# Start 

> docker-compose up -d


# Access 

Lookup the ip address of boot2docker

> docker-machine ip dev 

Replace ip address with the boot2docker ip address.

###### Mesos Master	

http://ip:5050

###### Marathon	

http://ip:8080

###### Chronos	

http://ip:4400
